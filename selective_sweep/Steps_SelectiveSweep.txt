Calculate nucleotide diversity:
	1. get the reference line randomly with the same number with the X or 3R insertion 
		perl getrandom.pl X_ind_75_exclude X_ind_75 X_ind_75_exclude_rand
	2. calculate PI in a window
		perl calInsertPi.pl X_ind_75_exclude_rand X_ind_75 X 2639224 50000 insertX_50K.pi  
	3. arrange files used for graphical presentation
		perl normal.pl Xref-50000.windowed.pi Xalt-50000.windowed.pi  X-50000.pi
	4. draw a dot-line plot to show the difference between the lines containing the insertion or not
		Rscript pi.R

Random sampling to verify the pattern is certain:
	1. random get 1000 sites from the same chr2.7Mb frquency synonymous site and recombination rate site and keep the refnum=altnum the output file is sum_X_1000 ; chr pos refpi altpi
 		perl simulate.pl 10000 
	2. draw a simulation plot
		Rscript sim.R
